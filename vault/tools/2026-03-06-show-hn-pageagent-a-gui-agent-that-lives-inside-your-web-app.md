## Summary

Open-source library that embeds AI agents directly into web frontends, allowing native DOM interaction and cross-page browser control

The 'inside-out' paradigm of embedding AI agents natively within web apps (rather than external bots) could unlock new possibilities for AI-powered user interfaces and automation within existing web applications
