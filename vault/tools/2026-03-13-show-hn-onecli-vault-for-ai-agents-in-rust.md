## Summary

OneCLI is a Rust-based proxy that securely manages API credentials for AI agents by storing real keys in an encrypted vault and providing placeholder tokens.

## Key Insight

Instead of restricting AI agent API access, the solution is giving agents secure access through credential proxying - they get placeholder keys that are swapped for real credentials at request time.
